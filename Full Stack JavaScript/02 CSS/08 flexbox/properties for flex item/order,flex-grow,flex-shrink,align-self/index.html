flex item properties: properties which can be applied on flex items

1) order: CSS property sets the order to lay out an item in a flex or grid container. Items in a container are sorted by ascending order value and then by their source code order.

Higher the order later it appears in flex container

3) The flex-shrink: CSS property sets the flex shrink factor of a flex item. If the size of all flex items is larger than the flex container, items shrink to fit according to flex-shrink.

if flex-direction = row then width will be affected

if if flex-direction = column height will be affected
