Comprehensive Guide: When to Use Em vs. Rem

1) How rem Units Translate to Pixel Values

* the pixel size they translate to depends on the font size of the root element of the page, i.e. the html element.

* For example, with a root element font size of 16px, 10rem would equate to 160px, i.e. 10 x 16 = 160.

* U can check value of font size of particular element in "computed" section of developer tool

2) How em Units Translate to Pixel Values

* note: It’s a somewhat widespread misconception that em units are relative to the font size of the parent element. In fact, as per the W3 spec, they are relative to the font size “of the element on which they are used”.

*Parent element font sizes can effect em values, but when that happens it’s solely because of inheritance.

3) vw & vh

* vw: View width. 1vw is 1% of the width of the viewport.

* vh: View height. 1vh is 1% of the height of the viewport.

vw & bh are set to 50 will result into box of width - 50% of viewport width and 50% of viewport height
